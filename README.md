# jizhicmsplugin
极致CMS的插件库，存放极致CMS各种插件 
## jizhicms_webapi   
火车头采集Web发布插件，解压后，把webapi放到A/exts/文件夹下面，火车头规则导入进火车头里面，后台开启插件并配置好登录密码即可使用     
## msgsafe  
留言提交安全插件，限制用户当天提交数      
## test  
官方插件演示版demo    
## adminsafe   
后台加强版安全登录插件    
## banip   
前台屏蔽IP访问插件   
## filemanage   
在线编辑模板插件   
## msgvercode   
前台留言加验证码插件      
## website   
多网址绑定网站并指定模板   
## update_1_3_to_1_4   
更新插件(仅从v1.3 更新到 v1.4) 
## update_1_4_to_1_5   
更新插件(仅从v1.4 更新到 v1.5)
